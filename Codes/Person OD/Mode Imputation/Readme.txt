The codes can be used to impute travel mode for mobile data locations.

DIRECTORY STRUCTURE:

The directory includes the following:

Distance_Calculation: The python code calculates the distance of points to multimodal network

Attribute_Extraction: The R code in this folder takes the point file from the Inputs folder and creates a trip file with all trip attributes, in the Inputs folder.

Prediction: The Prediction python code in this folder uses the output of Attribute_Extraction code, which is a trip file, and imputes the modes.The folder also includes model file and other dependencies.
