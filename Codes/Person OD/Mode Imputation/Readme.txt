The codes can be used to impute travel mode from location-based services data.

DIRECTORY STRUCTURE:

The directory includes the following:

Distance_Calculation: The python code calculates the distance of points to a multimodal network

Attribute_Extraction: The R code in this folder takes the point file and creates a trip file with all trip attributes.

Prediction: The Prediction python code in this folder uses the output of Attribute_Extraction code, which is a trip file, and imputes the modes.The folder also includes model file and other dependencies.
